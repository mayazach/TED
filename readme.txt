Η εργασία έχει υλοποιηθεί με Spring Framework και με βάση 
δεδομένων MySQL. Η μεταγλώττιση γίνεται με Maven.

Το αρχείο application-dev.yml περιέχει τα στοιχεία σύνδεσης με τη
βάση. Για να χρησιμοποιηθεί βάση σε άλλη διεύθυνση θα πρέπει να 
δημιουργηθεί ένα άλλο application-*.yml αρχείο με τις πληροφορίες
σύνδεσης της νέας βάσης και κάποιο άλλο όνομα στη θέση του * στο
όνομα αρχείου και στην ρύθμιση spring: profiles. Η επιλογή μεταξύ
των βάσεων γίνεται αλλάζοντας τη ρύθμιση spring: profiles: active
στο αρχείο application.yml. Τα .yml αρχεία βρίσκονται στο φάκελο
resources.

Οι εντολές για το στήσιμο της βάσης βρίσκονται στο αρχείο rooms.sql
στο φάκελο dbScripts.

Τα html,css,JSP και Javascript αρχεία βρίσκονται στο φάκελο webapp.

Η εφαρμογή τρέχει σαν Java application. H διεύθυνσή της είναι
https://localhost:8443/rooms/ (βρίσκεται και στο application.yml).

Το πρωτόκολλο SSL/TLS έχει χρησιμοποιηθεί σε όλη την εφαρμογή. Το 
αρχείο με τα κλειδιά περιλαμβάνεται στο φάκελο resources.
